# modx-xhrTools

Features:
- нативный js
- работа с состояниями
- загрузка файлов
- поддержка websokets
- выбор хранения параметров (cache, session, db)
- рекурсивный вызов
- поддержка очереди вызовов
- плагин для кастомизации ответа, данных
- отказ от имён?
- поддрежка различных масок
- стриминг медиа?
- загрузка страниц по ajax, SPA
